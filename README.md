# Disparo em Massa V4

Este projeto foi construído usando o design system da MKT Lab v2.

## Como Executar

Abra o terminal neste diretório e execute:

```bash
npm install
npm run dev
```

Este app possui uma interface que replica a estrutura para criar e editar lógicas de disparo com WhatsApp API, Google Sheets e CRM MKT Lab.
